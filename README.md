# Java Data Structures

Реализация структур данных на Java 11.

## Структуры данных

### LinkedList
- Односвязный список
- Двусвязный список (DoublyLinkedList)
- Кольцевой список (CircularLinkedList)

### Stack
- Реализация стека на основе массива
- Реализация стека на основе связного списка

### Queue
- Очередь (Queue)
- Двусторонняя очередь (Deque)

### Tree
- Бинарное дерево
- Бинарное дерево поиска (BST)

## Требования

- Java 11 или выше
- Maven 3.6+

# Запуск тестов
